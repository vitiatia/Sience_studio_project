# Анализ рынка экспорта калийных удобрений (HS-4 3104)

## О проекте

Данный проект представляет собой комплексное эмпирическое исследование мирового и российского экспорта калийных удобрений (товарная группа HS-4 3104). В работе проведён анализ структуры рынка, оценка сравнительных преимуществ стран, построены эконометрические модели (гравитационная модель Пуассона PPML и VAR/VECM) для выявления факторов экспорта и нереализованного потенциала, а также определены приоритетные целевые рынки для России.

**Ключевые результаты:**
- Россия обладает устойчивым сравнительным преимуществом (RCA > 1.5) и является вторым крупнейшим экспортёром калийных удобрений в мире
- Наиболее значимые факторы экспорта: ВВП стран-партнёров и наличие общей границы
- Нереализованный экспортный потенциал для России оценён в размере ≈18 млн долл. (приоритетные рынки — Казахстан и Чили)
- Анализ импульсных откликов (IRF) показал устойчивость российского экспорта калийных удобрений к внешним шокам (сходимость за 48 месяцев)

## Данные и источники

| Источник | Показатели | Период |
|----------|------------|--------|
| BACI (CEPII) | Двусторонний экспорт | 2013–2021 |
| World Bank WDI | ВВП стран | 2013–2024 |
| OEC / Atlas of Economic Complexity | Экспорт, PCI | 2018–2024 |
| UN Comtrade | Помесячный экспорт | 2013–2022 |
| CEPII Gravity | Расстояние, общая граница, язык, торговые соглашения | - |

## Используемые методы

- **PPML (Poisson Pseudo Maximum Likelihood)** — гравитационная модель для панельных данных с робастными стандартными ошибками HC3
- **VAR/VECM** — анализ временных рядов, тесты стационарности (ADF/KPSS), тест Йохансена на коинтеграцию, функции импульсного отклика (IRF) на 48 месяцев
- **Индексы** — RCA (выявленное сравнительное преимущество), HHI (концентрация Херфиндаля-Хиршмана), PCI (сложность продукта), ECI (экономическая сложность)

## Ограничения исследования

1. Отсутствие анализа конкурентной среды на целевых рынках (Канада, Германия, Беларусь)
2. Санкционные риски и их непредсказуемая динамика
3. Модель не учитывает возможное восстановление европейского спроса при смягчении логистических ограничений

## Направления для будущих исследований

- Макроэкономическое моделирование конкурентной среды на рынке калийных удобрений
- Прогнозирование поведения иностранных экспортёров при изменении цен на газ
- Включение в гравитационную модель переменных санкционной интенсивности
